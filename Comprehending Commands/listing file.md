# listing files
In this challenge we learnt about the ls command....which is used to list all the differnt file in a certain location. In this the name of the run was changed and we had to find
it using the ls command.
## Command Line
```
Connected!
hacker@commands~listing-files:~$ ls /challenge
9799-renamed-run-2289  DESCRIPTION.md
hacker@commands~listing-files:~$ /challenge/9799-renamed-run-2289
Yahaha, you found me! Here is your flag:
pwn.college{ot3qUR9qx--yvWFAHrGOE79908N.dhjM4QDLxcDN0czW}
```
